Concepto inicial

Gestión perpetua de cementerios impulsada por StarkNet

Visualiza filas alfabéticas, controla los derechos de administración y ofrece trazabilidad transparente a las familias. Este prototipo demuestra las capacidades que podemos presentar a los clientes para obtener retroalimentación temprana.

Qué ofrece el sistema

Orden alfabético garantizado

Los lotes se ordenan automáticamente por filas A-Z, evitando conflictos y manteniendo la consistencia con la normativa local.

Seguridad criptográfica

Cada titular queda protegido mediante compromisos Pedersen en la cadena de bloques de StarkNet, preservando la privacidad de los datos sensibles.

Operación 24/7

Diseñado para vigencia vitalicia con bitácoras auditables y reglas de acceso claras entre equipos municipales y concesionarios.

Flujo operativo sugerido

  1. Alta y verificación

    El administrador crea la fila y el lote, validando el orden alfabético antes de firmar la transacción.

  2. Generación de compromiso

    Se captura la información sensible del titular, generando un hash con Pedersen para almacenarlo en la blockchain.

  3. Entrega de constancias

    El sistema produce un comprobante digital verificable y un folio público con datos no sensibles para las familias.
